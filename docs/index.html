üîç Search Commands

Basic Search

pubmed search "fragile X syndrome" --limit 5 --json
Expected Output 
{"count": 8076, "ids": ["41651661", "41651254", ...], "query_translation": "..."}

Human-Readable Output

pubmed search "CRISPR" --limit 3 --human

Year Filter

pubmed search "autism intervention" --year 2024-2025 --limit 5 --json

Publication Type Filters

# Reviews only
pubmed search "diabetes treatment" --type review --limit 3 --json

# Clinical trials
pubmed search "COVID vaccine" --type trial --year 2024-2025 --limit 3 --json

# Meta-analyses
pubmed search "statin cardiovascular" --type meta-analysis --limit 3 --human

Sort Options

# Sort by date (newest first)
pubmed search "machine learning radiology" --sort date --limit 3 --json

# Sort by citations
pubmed search "CRISPR" --sort cited --limit 5 --human

CSV Export

pubmed search "FMRP" --limit 10 --csv results.csv

Special Characters & Boolean

# Greek characters
pubmed search 'IL-6 AND (TNF-Œ± OR TNF-alpha)' --limit 3 --json

# Quoted phrases
pubmed search '"machine learning" AND "deep learning"' --limit 3 --json

üìù Literature Synthesis

Note: Synthesis requires an LLM backend. Use --claude for Claude CLI or set LLM_API_KEY for OpenAI-compatible APIs.

Basic Synthesis

pubmed synth "FMRP synaptic function" --papers 3 --words 100 --claude

Single Paper Deep Dive

pubmed synth --pmid 41401891 --words 150 --claude

BibTeX + RIS Export

pubmed synth "ketamine depression" --papers 3 --bibtex refs.bib --ris refs.ris --claude

Word Document Output

pubmed synth "ADHD stimulant medications" --papers 3 --words 100 --docx review.docx --claude

Requires pandoc installed.

JSON Output (for agents)

pubmed synth "autism biomarkers" --papers 5 --claude --json

‚ùì Question Answering

Basic QA

pubmed qa "Does FMRP regulate synaptic plasticity?" --claude --json
Expected Output 
{
  "question": "Does FMRP regulate synaptic plasticity?",
  "answer": "yes",
  "confidence": 5,
  "strategy": "retrieval",
  "novel_detected": false,
  "source_pmids": ["41401891", "41276503", "41177899"],
  "context": "..."
}

Human-Readable Output

pubmed qa "Is metformin effective for type 2 diabetes?" --claude --human
Expected Output 
üß† Answer: YES
   Strategy: parametric
   Confidence: 10/10

Force Retrieval (Novel Questions)

pubmed qa "Does the 2025 RECOVER trial show paxlovid helps long COVID?" --retrieve --claude --json

Force Parametric (Skip Retrieval)

pubmed qa "Does aspirin reduce heart attack risk?" --parametric --claude --json

Custom Confidence Threshold

pubmed qa "Does lithium prevent suicide in bipolar disorder?" --confidence 9 --explain --claude --json

Higher threshold = more likely to trigger retrieval.
